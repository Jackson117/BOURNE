# 课题四图表示学习模型
该图表示学习模型使用层次化超图对多元异质信息进行建模，通过引入互注意力机制，可以有效学习网络中的异质信息，该表示学习模型是后续图异常检测的基础。

## 环境依赖
+ python==3.8.13
+ matplotlib==3.5.1
+ networkx==3.1
+ numpy==1.21.6
+ scipy==1.8.0
+ sklearn==0.24.1
+ tensorflow==1.15
+ tqdm==4.65.0

## 代码运行
### 层次超图构建
```
cd data
python process_cora.py
```

### 异质图表示学习
```
python exm_dblp.py --data coauthorship --dataset cora --depth 2 --dropout 0.1 --cuda True
```
